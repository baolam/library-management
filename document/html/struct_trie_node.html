Mô tả chi tiết

Một TrieNode dùng để lưu trữ dữ liệu.

Mảng các TrieNode con dùng để kết nối đến các kí tự khác.

Do việc tiến hành quản lí dữ liệu được cài đặt chính trên Id, nên cần phải tích hợp thêm khoá Id vào trong cấu trúc dữ liệu này.

Mặt khác do name có thể tồn tại trùng nhau nên Id ở đây là mảng.
